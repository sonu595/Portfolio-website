# Sonu Singh - Portfolio

My personal portfolio website built with React, Tailwind CSS, and Framer Motion.

## 🔗 Live Demo

View Portfolio =>  *(https://sonuportfoliowebsite.vercel.app/)*

## Built With

- React 19
- Tailwind CSS
- Framer Motion
- React Router DOM

## Features

- Fully responsive (mobile + desktop)
- Smooth scroll animations
- Interactive UI components
- Resume download option
- Social media links

## Pages

- **Home** - Hero section with background video + expertise section
- **About** - My journey, experience, and resume
- **Work** - Projects showcase *(coming soon)*
- **Contact** - Get in touch 

## 🚀 Run Locally

```bash
npm install
npm run dev
